【プロジェクト名】
Quiz Battle (4択クイズゲーム)test

【概要】
迫りくる敵をクイズで倒す、ブラウザベースの4択クイズゲームです。
制限時間内に正解を選んでステージを進行し、全5ステージのクリアを目指します。

【仕様】
1. 基本ルール
   - 全5ステージ（5問）。
   - 持ち時間は1問につき10秒。
   - プレイヤーの体力（HP）は3。
   - 3回ミス（時間切れ含む）するとゲームオーバー。
   - 5問すべて正解するとゲームクリア。

2. ゲームプレイ
   - **敵の接近**: 残り時間が減るにつれて、画面中央の敵画像が拡大（接近）します。
   - **回答**: 画面下部の4つの選択肢から回答します。
   - **正解時**: 
     - 「CORRECT!」と表示。
     - 解説モーダルが表示され、読むことができます。
     - 「NEXT STAGE」ボタンで次のステージへ進みます。
   - **不正解時**:
     - 画面が赤くフラッシュし、シェイク（振動）します。
     - HPが1減少します。
     - **正解は表示されません**（再挑戦が必要）。
     - 同じ問題に再度挑戦します（時間はリセット）。
   - **時間切れ**:
     - 不正解と同じ扱い（ダメージ＋再挑戦）。

3. コンテンツ（問題）
   - ユーザー指定のカスタム問題（FMKJ氏に関するクイズ）を5問搭載。

【技術スタック】
- HTML5
- CSS3 (Animations, Flexbox/Grid)
- JavaScript (Vanilla JS, No frameworks)

【ファイル構成】
- index.html   : ゲームのメイン画面構造
- style.css    : デザイン、アニメーション定義
- script.js    : ゲームロジック（タイマー、判定、進行管理）
- assets/      : 画像素材フォルダ
  - enemy1.png ～ enemy5.png : 各ステージの敵画像
  - background.png : 背景画像

【起動方法】
フォルダ内の `index.html` をWebブラウザ（Chrome, Edgeなど）で開いてください。

【特記事項】
- 画像アセットはAIによって生成されたオリジナルのものです。
- 別の環境やAIエージェントに引き継ぐ際は、このテキストファイルとソースコード一式を共有することで、仕様と構造を即座に把握可能です。
